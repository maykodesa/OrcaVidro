# 🪟 OrcaVidro

Sistema freelancer desenvolvido para vidraçarias com foco em geração de orçamentos, organização de pedidos e praticidade no atendimento.

Projeto totalmente desenvolvido em um único arquivo HTML utilizando HTML, CSS, JavaScript, AJAX e Canvas API.

---

## 🚀 Sobre o Projeto

O **OrcaVidro** é um sistema web criado para facilitar o processo de orçamento em vidraçarias, oferecendo rapidez, organização e uma interface moderna.

O projeto foi desenvolvido em estrutura simplificada utilizando tecnologias front-end integradas diretamente no HTML.

---

## ✨ Funcionalidades

- ✅ Geração de orçamentos
- ✅ Cadastro de clientes
- ✅ Controle de pedidos
- ✅ Atualizações dinâmicas com AJAX
- ✅ Interface responsiva
- ✅ Utilização de Canvas API
- ✅ Sistema leve e rápido
- ✅ Organização de serviços

## ⚙️ Como Executar

### Clone o repositório

```bash
git clone https://github.com/maykodesa/OrcaVidro.git
```

### Acesse a pasta

```bash
cd OrcaVidro
```

### Execute o sistema

Abra o arquivo:

```bash
index.html
```

Ou utilize extensões/servidores locais como:

- Live Server
- XAMPP
- WAMP
- Laragon
